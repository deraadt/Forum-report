Forum activity and participation report

This version of the report is compatible with Moodle 2.2 or later.

To install from a Zip file:
1. Extract the Zip file to reports/forums
2. Visit the Notifications page (Settings > Site administration > Notifications)
3. Report should appear and be installed
4. Access the report from within a course (Navigation > Courses > CourseXXX > Reports > Forum overview)

To install using Git
1. Type this command in the root of your Moodle install
      git clone git://github.com/deraadt/Forum-report.git report/forum
2. Visit the Notifications page (Settings > Site administration > Notifications)
3. Report should appear and be installed
4. Access the report from within a course (Navigation > Courses > CourseXXX > Reports > Forum overview)
